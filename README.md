## 最小定義

- E = Expression（表現）
- R = Recordable
- I = Interpretive
- V = 総評
- θ = 向き（R寄りか／I寄りかを見るための見方）

## 位置付け

`E = R + I + V(θ)` は、表現整理フレーム『ＲＩＶ－ＴＨＥＴＡ』を覚えやすくするための数学的比喩です。  
方式名『複素構造評価』 / Complex Structure Evaluation の正式な定義式ではありません。

## ＲＩＶ－ＴＨＥＴＡの位置づけ

表現整理フレーム『ＲＩＶ－ＴＨＥＴＡ』は、日常の話し方・書き方を、R / I / V / θ / E に分けて整えるための入口です。

本フレームは、方式名『複素構造評価』そのものの全実装手順、算出手順、運用基準、監査手順を公開するものではありません。

本フレームでいう E = R + I + V(θ) は、表現整理フレーム『ＲＩＶ－ＴＨＥＴＡ』を要約して示す代表式・覚え方であり、方式名『複素構造評価』の正式な定義式そのものではありません。

## E と次工程への接続

本フレームでいう E は、日常文面として完結する場合もあれば、既存様式、既存手法、会議、記録、次の判断材料、または方式名『複素構造評価』へ渡す中間出力になる場合もあります。

そのため、本フレームの利用は、単発の「入力 → 本フレーム → 最終文面」に限られません。次工程への受け渡し、既存手法への再接続、改善サイクルへの戻り等を含む、開いた接続として扱われる場合があります。

## 補足

- 本記載における R / I は Real / Imaginary ではありません。
- 本記載における V は総評を表す記号であり、ベクトルを表すものではありません。
- 方式名『複素構造評価』との接続時には、必要に応じて `v = (R, I)` を評価ベクトルとして扱います。
- 本記載における θ は、まず「向き」を表すものとして置いています。方式名『複素構造評価』との接続時には、必要に応じて `v = (R, I)` の向きとして読みます。
- 『ＲＩＶ－ＴＨＥＴＡ』および方式名『複素構造評価』 / Complex Structure Evaluation は、本文中では名称として用いています。これらの名称を一般名詞的に用いる趣旨ではありません。
