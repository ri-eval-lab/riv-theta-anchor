# 表現整理フレーム『ＲＩＶ－ＴＨＥＴＡ』

RI Eval Lab

本リポジトリは、表現整理フレーム『ＲＩＶ－ＴＨＥＴＡ』および

`E = R + I + V(θ)`

という数学的比喩について、公開時点の最小限のデジタル証跡を残すための公開アンカーです。

## 記録対象
- 名称：表現整理フレーム『ＲＩＶ－ＴＨＥＴＡ』
- 読み：リヴシータ
- 出所表示：RI Eval Lab
- 最小式：`E = R + I + V(θ)`

## 最小定義
- E = Expression（表現）
- R = Recordable
- I = Interpretive
- V = 総評
- θ = 向き（R寄りか／I寄りかを見るための見方）

## 位置付け
`E = R + I + V(θ)` は、表現整理フレーム『ＲＩＶ－ＴＨＥＴＡ』を覚えやすくするための数学的比喩です。  
方式名『複素構造評価』 / Complex Structure Evaluation の正式な定義式ではありません。

## 補足
- 本記載における R / I は Real / Imaginary ではありません。
- 本記載における V は総評を表す記号であり、ベクトルを表すものではありません。
- 本方式との接続時には、必要に応じて `v=(R,I)` を評価ベクトルとして扱います。
- 本記載における θ は、まず「向き」を表すものとして置いています。本方式との接続時には、必要に応じて `v=(R,I)` の向きとして読みます。
- 『ＲＩＶ－ＴＨＥＴＡ』および方式名『複素構造評価』 / Complex Structure Evaluation は、本文中では名称として用いています。これらの名称を一般名詞的に用いる趣旨ではありません。
